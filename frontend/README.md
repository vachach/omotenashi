# Omotenashi Academy Frontend

Responsive landing page for Omotenashi Academy, built with React, Vite, Tailwind CSS, and Lucide React icons.

## Live Site

Production deployment:

https://frontend-omega-one-24.vercel.app/

## Tech Stack

- React 19
- Vite 7
- Tailwind CSS
- Lucide React
- ESLint

## Getting Started

Install dependencies:

```bash
npm install
```

Run the local development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Run lint checks:

```bash
npm run lint
```

## Vercel Deployment

Recommended Vercel settings:

- Root Directory: `frontend`
- Framework Preset: `Vite`
- Build Command: `npm run build`
- Output Directory: `dist`

## Project Notes

The site is optimized for desktop, tablet, and mobile layouts. The design uses the existing Omotenashi visual assets from `public/`, including the logo, samurai hero image, and Japanese cultural illustrations.
